# Sundaramahalingam Palanisamy's Portfolio

A React portfolio with light yellow background and white theme, hosted on GitHub Pages.

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build for GitHub Pages

```bash
npm run build
```

## Deploy to GitHub Pages

1. Push this project to your `sundar.github.io` repository
2. Run: `npm run deploy`
3. Your site will be live at https://sundar.github.io

## Manual Deploy

1. Run `npm run build`
2. Copy contents of `dist/` folder to your `sundar.github.io` repo
3. Push to GitHub
